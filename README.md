# OPT_PHP_DanielMolina
Este repositorio contiene unos scripts básicos sobre html, javascript y php para simular el registro y administración de los empleados en una empresa.

El html compone el formulario.
EL javascript proporciona dinamismo al html para transformar el input de la sede en uno dinámico en base a la provincia elegida.
EL php revisa los datos recogidos por el formulario y revisa sus formatos.
